Quick Instructions

To start loading your deck list in the Deck Importer just type or paste the names of the cards you want in the Card List (one per line).
The Card list supports the majority of M:TG deck formats: MTG Arena; MTGO; XMage; MTG Goldfish (all export); TappedOut (all export, but CSV); DeckStats.net; ".dec" files.

Each line in the Card list is processed immediately, and any recognized entry will appear in the Deck table.
Prepend to each card line the number of copies to import (default: 1). (e.g. "4 Giant Growth" or "4x Giant Growth"), and also include the code of an M:TG set before or after the card name to select a specific card art.
Lines starting with # or // will be interpreted as comments, and so skipped in the deck list parsing.

Examples

10x Islands MMQ 2 (F): 10 FOIL "Island" from Set "Mercadian Masques", Art Index 2. The "(F)" at the end of the line denotes a foil card
4 Forsaken City (PLS) 139: 4 "Forsaken City" from Planeshift ("PLS"), Collector Nr. "139".

This is an exemplar entry in MTGArena format as exported by TappedOut.
18 Forest <348> [TE]: 18 "Forest" from Tempest (TE), Collector Nr. "348".

This is an exemplar entry in MTGArena deck format as exported by MTGGoldfish - also using the set code of MTGO for Tempest set. The Collector Number in angular brackets can be used to select a specific Card Art.
